# what is vocabulary in tokenization ?
- Vocabulary is the set of all unique tokens that a tokenizer knows how to process.

# What is token?
    - A token is a sequence of characters that is treated as a single unit by the tokenizer.

# What is tokenization?
    - Tokenization is the process of splitting a text into tokens.


# What is GPT?
    - GPT (Generative Pre-trained Transformer) is a type of language model developed by OpenAI that uses a transformer architecture to generate human-like text based on the input it receives. It is pre-trained on a large corpus of text data and can be fine-tuned for various natural language processing tasks.

# What is BERT?
    - BERT (Bidirectional Encoder Representations from Transformers) is a transformer-based language model developed by `Google that is designed to understand the context of words in a sentence by looking at both the left and right sides of the word`. It is pre-trained on a large corpus of text data and can be fine-tuned for various natural language processing tasks, such as question answering and sentiment analysis.
